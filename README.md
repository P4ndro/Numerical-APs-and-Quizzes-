# Numerical Analysis Projects

A collection of numerical analysis and computational mathematics assignments and projects.

## üìÇ Project Structure

### AP1 - Distance Metrics and Norms
Python implementation exploring different distance metrics and unit ball visualizations in 2D and 3D spaces.

**Key Features:**
- Distance comparison analysis
- Unit ball visualizations
- 2D and 3D plots

**Files:**
- `problem_1_1.py` - Main implementation
- Visualization outputs: `unit_balls_2d.png`, `unit_balls_3d.png`, `distance_comparison.png`

### AP2 - Clustering Analysis
Steam dataset clustering analysis using various clustering algorithms.

**Key Features:**
- K-means clustering
- Cluster visualization
- Optimal K analysis

**Files:**
- `steam_clustering.py` - Clustering implementation
- Input data: `steam.csv`
- Results: `steam_clustering_results.csv`, `cluster_analysis.csv`
- Visualizations: `clusters_2d.png`, `clusters_3d.png`, `optimal_k_analysis.png`

### AP3 - Function Analysis
Numerical analysis of 1D and 2D functions with comprehensive visualization and tabulation.

**Key Features:**
- Function evaluation and analysis
- Automated visualization
- CSV output for numerical results

**Files:**
- `problem_3_1.py` - Main implementation
- Outputs: `1d_function_*.png`, `2d_function_*.png`
- Data tables: `table_1d_*.csv`, `table_2d_*.csv`, `summary_*.csv`

### AP4 - Tornado Shelter Voronoi Diagrams
Voronoi diagram analysis for tornado shelter placement optimization.

**Key Features:**
- Voronoi diagram generation
- Spatial analysis
- Demo implementations

**Files:**
- `tornado_shelter_voronoi.py` - Main implementation
- `demo_no_save.py` - Demo script
- `report_template.md` - Report template

### quiz2 - Assessment Tasks
Collection of quiz implementations covering various numerical methods.

**Files:**
- `task3.py`, `task4.py`, `task5.py`, `task6.py` - Individual tasks
- `task6_plot.png` - Visualization output
